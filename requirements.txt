MEDI CARE - ON-DEMAND HOME HEALTHCARE DISPATCH SYSTEM
Requirements & Technical Specifications
================================================================================

PROJECT OVERVIEW
================================================================================
An on-demand home-to-home healthcare service dashboard (like Uber for healthcare)
that manages mobile healthcare providers who travel to patients' homes with 
real-time tracking, request assignment, and dispatch functionality.

================================================================================

INSTALLATION
================================================================================
1. Install dependencies:
   npm install
   # or
   pnpm install

2. Run development server:
   npm run dev
   # or
   pnpm dev

3. Build for production:
   npm run build
   # or
   pnpm build

PAGES IMPLEMENTED
================================================================================
1. Dashboard
   - Stats cards (Total Requests, Active Providers, Avg Response Time, Completion Rate)
   - Interactive map with provider locations
   - Live requests panel
   - Online providers sidebar

2. Patients
   - Patient list with search and filters
   - Visit history tracking
   - Health records
   - Upcoming appointments
   - Patient profile details

3. Providers
   - Provider directory
   - Real-time location tracking
   - Vehicle information
   - Service areas
   - Availability status
   - Performance metrics

4. Requests
   - Request queue management
   - Status filtering (Pending/Assigned/In Progress/Completed/Cancelled)
   - Urgency levels (Emergency/Urgent/Normal)
   - Provider assignment
   - Detailed request tracking
   - Request timeline

5. Reports
   - Key performance metrics
   - Daily visits chart
   - Service type distribution
   - Revenue trends
   - Peak hours analysis
   - Provider performance rankings
   - Export functionality
